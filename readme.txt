CONFIGURATION
--
Server: Network_Project\Network_Project\Server\bin\config.xml
-> Fill in the file with the listening port

Client: Network_Project\Network_Project\Client\bin\config.xml
-> Fill in the file with the IP address and port of the server

EXECUTION
--
Server: run the file Network_Project\Network_Project\Server\bin\Server.exe

Client: run the file Network_Project\Network_Project\Client\bin\Client.exe
